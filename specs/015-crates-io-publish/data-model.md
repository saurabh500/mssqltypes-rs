## Entity: Cargo.toml `[package]` Metadata

The central entity is the `[package]` section of `Cargo.toml`. This is consumed by `cargo publish` and rendered on crates.io.

### Fields (Current → Target)

| Field | Current Value | Target Value | Validation Rules |
|-------|--------------|-------------|-----------------|
| `name` | `"mssqltypes"` | `"mssqltypes"` (unchanged) | Must be unique on crates.io; lowercase, alphanumeric + hyphens |
| `version` | `"0.1.0"` | `"0.1.0"` (unchanged) | SemVer format |
| `edition` | `"2024"` | `"2024"` (unchanged) | Valid Rust edition |
| `description` | _(missing)_ | Concise summary | Required; plain text; ≤1000 chars |
| `license` | _(missing)_ | `"MIT"` | Required; must match LICENSE file; SPDX identifier |
| `repository` | _(missing)_ | GitHub URL | Valid URL |
| `readme` | _(missing)_ | `"README.md"` | Must point to existing file |
| `keywords` | _(missing)_ | 5 terms | Max 5; each ≤20 ASCII chars |
| `categories` | _(missing)_ | 2 slugs | Max 5; must be valid crates.io category slugs |
| `rust-version` | _(missing)_ | `"1.85"` | Two-component version; must support chosen edition |
| `exclude` | _(missing)_ | 3 patterns | Glob patterns; must not exclude `src/`, `Cargo.toml`, `LICENSE`, `README.md` |

### Relationships

```text
Cargo.toml [package]
├── references → LICENSE (via `license` field)
├── references → README.md (via `readme` field)  
├── constrains → package file list (via `exclude` field)
└── constrains → minimum toolchain (via `rust-version` field)
```

## Entity: Crate-Level Documentation (`//!` in lib.rs)

Content rendered as the docs.rs landing page for the crate.

### Structure

| Section | Content | Validation |
|---------|---------|-----------|
| Title | `# mssqltypes` | Must match crate name |
| Description | One-paragraph overview | Must describe purpose and key features |
| Type table | All 14 types with SQL/Rust mapping | Must list all public types |
| Quick Start | Code example with `SqlInt32` | Must compile as a doc-test |
| Feature flags | Optional `serde` support | Must document available features |

### Relationships

```text
src/lib.rs //! docs
├── references → all 14 SqlType structs
├── references → SqlTypeError enum
├── produces → doc-tests (verified by `cargo test --doc`)
└── rendered on → docs.rs landing page
```
